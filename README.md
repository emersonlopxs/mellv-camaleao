# MellV

### Requerimentos

1 - Node >= 10

2 - Yarn

### usando o yarn ao inves to npm

1 - `npm install -g yarn`

### Como rodar o app

1 - `yarn install`

2 - `yarn start`

O aplicativo estará disponivel em http://localhost:3000/about/
